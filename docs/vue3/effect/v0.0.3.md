##### vue3 基本上是这么干的：

通过 `Proxy` 对象，我们可以拦截对象的一些原子操作，如 `getter` 、 `setter` 等。

现在，我们通过拦截对象的 `setter` 操作，当对象的属性值被赋值时，便去执行 `effect` 方法！
这样就可以做到，当对象的属性值改变时，视图就跟着更新了。
