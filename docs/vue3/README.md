### 简述

对于前端框架，如果只停留在知其然而不知其所以然的层次，那当工作中遇到一些问题时，就会非常被动。

如果我们知道框架的实现原理，不光问题排查的更快，而且也有助于我们写出更加优雅的代码。

若我们不光知道其实现原理，还凭着自己的理解，手码了一遍，那说明了什么？

说明你已经很强，源码已经难不住你了，地球已经不适合你居住了！

当然，我是边看书边手码的。

还有就是，我希望大家看完之后，能轻松手码实现一遍轮子！
